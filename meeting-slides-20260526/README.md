# 第1回 異業種交流会×企画ミーティング スライド

`index.html` だけで動く共有用HTMLスライドです。

## ローカル確認

`index.html` をブラウザで開くと確認できます。

## 共有方法

### Vercelで公開

Vercel CLI にログイン済みなら、このフォルダで以下を実行します。

```bash
vercel --prod
```

公開後に発行されるURLを共有してください。ページ下部の共有アイコンからもURL共有・コピーできます。

### Netlify Dropで公開

`shareable-meeting-slides` フォルダを ZIP 化するか、そのまま Netlify Drop にドラッグすると公開できます。

### ファイルとして共有

`index.html` をそのまま送ることもできます。Google Fonts、Font Awesome、QRコード画像を読み込むため、閲覧時にはインターネット接続が必要です。
